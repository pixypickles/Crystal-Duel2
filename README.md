# Crystal Duel 2 — Mobile v2

## 今回の修正

- 支給された4枚のイラストを透過処理し、ゲーム内キャラクターとして確実に表示
- キャラクター表示を大きく調整
- 弾速を約37%低下
- 攻撃ボタンの長押しでチャージ弾
  - 0.7秒長押しでダメージ2
  - 杖先が点滅
- ジャストガードを追加
  - ガード開始から0.18秒以内に弾を受けると反射
  - 通常ガードは弾を消去
- 反射弾は相手側へ飛び、通常の弾として衝突・クリスタル攻撃が可能

## 操作

- ▲ / ▼：5レーンを上下移動
- ↗ / → / ↘：短押しで通常弾、長押しでチャージ弾
- GUARD：押している間ガード
- ガード直後に弾を受けるとジャストガード反射
